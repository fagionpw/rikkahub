Assistant Context Menu & Popup

Select any text on this page, open the context menu, and choose Send to assistant to open the popup interface.

Demo Article

This sandbox shows how a Google Translate style context menu entry can forward selected text into a floating assistant widget. The popup mirrors the Google Search widget layout, consolidating assistant selection, freeform text entry, voice capture, and image capture inside a single interface. Responses from the assistant appear directly below the input area so the conversation stays focused.

Try highlighting this paragraph, open the context menu, and choose the assistant option. You can also open the popup manually with the button below for quick testing.
